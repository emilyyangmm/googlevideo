# 小红书全自动发布技能

**一键完成：** 标题生成 → 文案撰写 → AI 配图 → 发布笔记

## 工作流程

### 第 1 步：生成爆款标题（5 个候选）

根据用户提供的主题，使用 6 大爆款公式生成标题：

| 公式 | 示例 |
|------|------|
| 数字型 | "3 天从 0 到 1，我的 AI 生图网站上线了！" |
| 痛点型 | "做 AI 网站踩的 3 个坑，早知道就好了😭" |
| 好奇型 | "闺蜜问我怎么靠 AI 生图月入过万的..." |
| 对比型 | "用 AI 生图前 vs 后，同事都问我是不是报班了" |
| 权威型 | "15 年采购经理推荐的 AI 工具，真香！" |
| 情绪型 | "后悔没早点做！AI 生图真能搞钱啊" |

**标题规则：**
- ✅ 长度 < 22 字
- ✅ 必须有 emoji
- ✅ 无广告法禁用词
- ✅ 无 AI 味

---

### 第 2 步：撰写笔记正文

**正文结构：**

```
【开头钩子】（前 50 字抓住注意力）
- 痛点/需求/好奇心

【主体内容】（分点阐述，每点 1-2 句）
- 点 1：核心信息
- 点 2：核心信息
- 点 3：核心信息

【结尾互动】（引导评论/收藏）
- 提问/求助/共鸣

【Tags】（5-10 个相关标签）
#AI 绘画 #副业搞钱 #打工人副业 #AI 工具 #效率神器
```

**正文规则：**
- ✅ 每段 1-3 句，短句为主
- ✅ 每段 1-2 个 emoji
- ✅ 总字数 300-800 字
- ✅ 口语化、有情绪

---

### 第 3 步：生成 AI 配图（3-4 张）

使用 **nano-banana-pro**（Gemini 生图）生成配图：

**封面图：**
- 尺寸：3:4（1080×1440）
- 内容：主题相关 + 大标题文字
- 风格：根据内容类别选择

**正文配图（2-3 张）：**
- 尺寸：3:4
- 内容：展示关键信息/步骤/效果

**生图 Prompt 公式：**
```
[主体] + [场景] + [风格] + [色调] + [构图]
示例：一个年轻女孩在使用电脑，现代办公室，简约风格，明亮色调，中景
```

---

### 第 4 步：发布笔记

使用 **xhs-mcp** 发布到小红书：

```bash
python scripts/xhs_client.py publish "<标题>" "<正文>" "<图片 1>,<图片 2>,<图片 3>"
```

**前置条件：**
- 配置小红书 Cookie（.env 文件）
- 或运行 MCP 服务器扫码登录

---

## 使用方式

**用户输入：**
```
主题：AI 生图网站教程
目标人群：想搞副业的打工人
核心卖点：3 天从 0 到 1 上线，已有 69 名用户
```

**输出：**
1. 5 个爆款标题候选
2. 完整笔记正文（带 emoji 和 Tags）
3. 3-4 张 AI 生成的配图
4. 发布确认（用户确认后发布）

## 注意事项

1. **标题** - 生成 5 个让用户选，或自动选最佳
2. **配图** - 生成后让用户确认，不满意可重绘
3. **发布** - 必须用户确认后才发布
4. **Cookie** - 需定期更新（有效期约 7 天）
