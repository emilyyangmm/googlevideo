# 小红书全自动发布技能 v2.0 🦞

> **虾王比赛参赛作品** - 热点驱动 + 用户主导 + 全流程自动化

---

## 🎯 技能亮点

1. **热点搜集** - 自动抓取小红书实时热榜
2. **汇报机制** - 整理热点向用户汇报
3. **用户主导** - 按用户意见创作，不擅自发布
4. **全流程自动化** - 确认后一键完成所有步骤

---

## 🚀 快速开始

### 1. 搜集热点

```bash
npm run hot
# 或
node scripts/xhs-hot.js 20
```

### 2. 创作内容

```bash
npm run create "AI 绘画网站"
# 或
node scripts/xhs-publish.js --create "AI 绘画网站"
```

### 3. 发布笔记

用户确认后：

```bash
npm run publish
```

---

## 📋 完整工作流

```
1. 搜集热点 → 2. 汇报给用户 → 3. 等待意见 → 
4. 生成内容 → 5. 用户确认 → 6. 自动发布
```

## 🔧 配置

### 环境变量

```bash
# .env
XHS_API_KEY=your_api_key
XHS_COOKIE=your_cookie
```

### MCP 服务器

确保小红书 MCP 服务运行在 `localhost:18060`
