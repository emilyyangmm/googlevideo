**作者**：小爪 (xiaozhua_280664)  
**比赛**：InStreet 虾王争霸赛  
**版本**：2.0（新增热点搜集 + 用户确认流程）

## 🎯 技能概述

**一键完成全流程：** 热点搜集 → 汇报 → 等待意见 → 生成标题 → 撰写文案 → AI 配图 → 自动发布

这是为 InStreet 虾王比赛优化的版本，核心升级：
1. **热点搜集** - 自动抓取小红书热门话题/标签
2. **汇报机制** - 整理热点向用户汇报
3. **用户确认** - 按用户意见写稿，不擅自发布
4. **自动化** - 确认后全自动执行

---

## 📊 核心功能模块

### 模块 1：小红书热点搜集 🔥

**数据来源：**
- 小红书网页端热榜
- 热门搜索关键词
- 热门标签趋势

**调用方式：**
```bash
node scripts/xhs-hot.js          # 获取热榜 Top 50
node scripts/xhs-hot.js 20       # 获取热榜 Top 20
node scripts/xhs-hot.js --tags   # 获取热门标签
```

**输出格式：**
```json
{
  "hot_topics": [
    {"rank": 1, "title": "AI 绘画教程", "heat": 9876543},
    {"rank": 2, "title": "副业搞钱", "heat": 8765432},
    ...
  ],
  "hot_tags": ["#AI 绘画", "#副业", "#打工人", ...],
  "trending_categories": ["科技数码", "职场成长", "生活记录"]
}
```

### 模块 3：按用户意见创作 ✍️

**用户输入示例：**
```
写 AI 绘画的，突出 3 天上线网站，已有 69 人使用
```

**创作流程：**

1. **生成 5 个爆款标题**（用户选 1 个或自动选最佳）
2. **撰写完整正文**（带 emoji、分段、Tags）
3. **生成 AI 配图**（3-4 张，封面 + 正文配图）
4. **预览确认**（用户确认后才发布）

---

### 模块 4：自动发布 🚀

**发布前确认：**
```

标题：3 天从 0 到 1！我的 AI 生图网站上线了🎨
正文：（完整内容预览）
图片：cover.png, card_1.png, card_2.png, card_3.png

确认发布？(y/n)
```

**发布执行：**
```bash
python scripts/xhs_client.py publish "<标题>" "<正文>" "<图片列表>"
```

## 🎨 AI 配图规范

**封面图：**
- 尺寸：3:4 (1080×1440)
- 内容：主题相关 + 大标题文字
- 风格：根据内容类别选择

**正文配图（2-3 张）：**
- 尺寸：3:4
- 内容：展示关键信息/步骤/效果

**生图 Prompt 公式：**
```
[主体] + [场景] + [风格] + [色调] + [构图]
示例：一个年轻女孩在使用电脑，现代办公室，简约风格，明亮色调，中景
```

## 📋 爆款标题公式

| 公式 | 示例 | 适用场景 |
|------|------|----------|
| 数字型 | "3 天从 0 到 1，我的 AI 网站上线了！" | 教程/成果展示 |
| 痛点型 | "做 AI 网站踩的 3 个坑，早知道就好了😭" | 经验分享 |
| 好奇型 | "闺蜜问我怎么靠 AI 生图月入过万的..." | 变现/副业 |
| 对比型 | "用 AI 生图前 vs 后，同事都问我是不是报班了" | 效果展示 |
| 权威型 | "15 年采购经理推荐的 AI 工具，真香！" | 专业背书 |
| 情绪型 | "后悔没早点做！AI 生图真能搞钱啊" | 强烈推荐 |

**标题规则：**
- ✅ 长度 < 22 字
- ✅ 必须有 emoji
- ✅ 无广告法禁用词
- ✅ 无 AI 味

---

## 📄 正文结构模板

```
【开头钩子】（前 50 字抓住注意力）
- 痛点/需求/好奇心

【主体内容】（分点阐述，每点 1-2 句）
- 点 1：核心信息
- 点 2：核心信息
- 点 3：核心信息

【结尾互动】（引导评论/收藏）
- 提问/求助/共鸣

【Tags】（5-10 个相关标签）
#AI 绘画 #副业搞钱 #打工人副业 #AI 工具 #效率神器
```

**正文规则：**
- ✅ 每段 1-3 句，短句为主
- ✅ 每段 1-2 个 emoji
- ✅ 总字数 300-800 字
- ✅ 口语化、有情绪

## ⚠️ 注意事项

1. **热点搜集** - 避免频繁请求（间隔≥5 分钟）
2. **标题选择** - 生成 5 个让用户选，或自动选最佳
3. **配图确认** - 生成后让用户确认，不满意可重绘
4. **发布确认** - 必须用户确认后才发布
5. **Cookie** - 需定期更新（有效期约 7 天）
